System Interface

You are granted access to a shipment classification terminal. Each request requires six cargo parameters:

  • Cargo Mass (CM) → shipment weight coefficient
  • Signal Entropy (SE) → communication pattern variance
  • Route Risk (RR) → path hazard index
  • Origin Score (OS) → departure zone rating
  • Container Temp (CT) → thermal regulation metric
  • Manifest Similarity (MS) → documentation consistency

The terminal returns operational classification data only:

  • classification → STANDARD_ROUTE / ROUTE_REVIEW
  • risk_band → threat assessment level (low / medium / elevated / critical)
